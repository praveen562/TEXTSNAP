# 📸 TextSnap

**TextSnap** is a smart text summarization web app that helps you condense lengthy content into concise summaries with ease. It also keeps a history of your summaries for later reference!

## 🚀 Features

- ✨ **Summarize Text**: Paste any content and instantly get a summarized version.
- 💾 **Download Option**: Save your summary as a `.txt` file with one click.
- 🕓 **History Page**: Automatically saves all summaries for future reference.
- 🏠 **Interactive Home Page**: Stylish landing page explaining the project with a "Check Now" button.
- 🔄 **Smooth Navigation**: Easily switch between Home, Summarizer, and History.
- 🎨 **Modern UI**: Clean and colorful design with fonts, icons, and responsive layout.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Font Awesome
- **Backend**: Python, Flask
- **Data Storage**: CSV (`history.csv`)
